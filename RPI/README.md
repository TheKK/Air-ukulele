Air-ukulele(RaspberryPi side)
===========================

Codes for rapberryPi to handle sound processing

Requirement
-----------

- pkg-config
- openAL
- freealut
- [gpio-admin](https://github.com/quick2wire/quick2wire-gpio-admin)

Installation
------------

Just type `make`
